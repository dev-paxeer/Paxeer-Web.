Frontend Integration Guide.

A comprehensive guide for integrating the PaxLend Protocol API, covering vaults, user credit, and protocol statistics.

üåê Base API URL

https://lending-api.paxeer.app

üìù Overview

This document provides comprehensive API documentation for integrating PaxLend Protocol. The API provides real-time vault data, user credit scoring, and lending protocol statistics.

Authentication

All endpoints are currently public and do not require authentication.

Response & Error Format

All successful API responses are wrapped in a standard format. Errors will return a `detail` and `status_code`.

// Success
{
  "data": {...},
  "timestamp": "2025-08-28T07:43:55Z",
  "status": "success"
}

// Error
{
  "detail": "Error message",
  "status_code": 400
}

async function fetchVaults() {
  const response = await fetch('https://lending-api.paxeer.app/api/v1/vaults');
  const data = await response.json();
  return data.vaults;
}

Performance: Caching & Batching

Implement a simple client-side cache to reduce redundant API calls and use `Promise.all` to fetch initial dashboard data concurrently for optimal performance.

const fetchDashboardData = async (userAddress) => {
  const [vaults, stats, credit] = await Promise.all([
    fetch('/api/v1/vaults'),
    fetch('/api/v1/protocol/stats'),
    fetch(`/api/v1/user/${userAddress}/credit`)
  ]);
  return { vaults, stats, credit };
};
